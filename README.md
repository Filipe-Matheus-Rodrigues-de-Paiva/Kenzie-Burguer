# Kenzie-Burguer
Burguer shop  - products showcase. A typecript + react application

Technologies/libraries
1. ReactJS with context API;
2. Axios;
3. Typescript;
4. Styled-Components (CSS-in-JS);
5. Toast;
6. Zod;
7. Hook-form;
8. React-Router-Dom;
9. React-icons.

Features
1. Login;
2. Signup;
3. Logout;
4. Save User preferences (localStorage);
5. add To cart;
6. delete products.
